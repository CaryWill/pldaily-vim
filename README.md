## My Vim Config

![](https://github.com/CaryWill/pldaily-vim/blob/master/screenshot.png?raw=true)

### 安装配置

> 克隆项目

```shell
git clone https://github.com/CaryWill/pldaily-vim.git ~/.vim
```

> 执行 shell

```shell
cd ~/.vim && sh bootstrap.sh
```

### Coc.nvim 插件

> 安装

```vim
:CocInstall coc-json coc-html coc-css coc-tsserver
```

> 查看

```vim
:CocList extensions
```

### 编辑配置文件

`:PlugInstall` 安装配置
